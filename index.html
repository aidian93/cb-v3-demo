Easy to use interface for Streamers

Project Overview

This project aims to provide streamers with an easy to use interface to launch and manage Chaturbate apps such as Tip Menus, Lovense, Tip Goals, and more with the aim of enhancing overall streamer and user experiences on the platform. Built using Python, SQL Server, MQTT, and Ignition, the project focuses on ease of use and high customization allowing for more complex apps to be built and used on the platform.

Technical Overview

This app was built using the following technologies:

  • Python – Core logic and app functionalities
  • SQL Server – Database management and storage
  • MQTT – Real-time communication between apps and devices
  • Ignition – User interface and script/app control visualization
